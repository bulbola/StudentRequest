# Частые проблемы и способы решения

Документ содержит типовые проблемы, которые могут возникнуть при запуске и демонстрации проекта StudentRequest.

## Python не найден

Если команда:

py --version

не показывает версию Python, необходимо установить Python и добавить его в PATH.

## Git не найден

Если команда:

git --version

не работает, необходимо установить Git for Windows и перезапустить терминал.

## Flask не найден

Если команда запуска Flask не работает, нужно установить зависимости:

py -m pip install -r requirements.txt

После этого приложение можно запускать командой:

py -m flask --app run.py run

## База данных не создана

Если приложение запускается, но данные отсутствуют, нужно выполнить:

py -m flask --app run.py init-db
py -m flask --app run.py seed-db

## Сервер уже запущен

Если терминал не принимает команды, возможно, в нем запущен Flask-сервер.

Для остановки сервера используется сочетание клавиш:

Ctrl + C

## Ошибка доступа к виртуальному окружению

Если PowerShell запрещает запуск activate, можно выполнять команды без активации окружения через py -m.

Например:

py -m flask --app run.py run
py -m pytest

## Вывод

Данный документ помогает быстро устранить основные проблемы при запуске и проверке проекта.
