# Итоговая сводка тестирования

Документ описывает результаты проверки StudentRequest перед сдачей проекта.

## Проверяемые элементы

В ходе проверки необходимо убедиться, что работают:

- главная страница;
- страница входа;
- страница справки;
- личный кабинет студента;
- форма создания заявки;
- панель сотрудника;
- страница обработки заявки;
- административная панель;
- список пользователей;
- список категорий;
- выгрузка XLSX-отчета.

## Проверка ролей

### Студент

Проверяется возможность входа, просмотра личного кабинета и создания заявки.

### Сотрудник

Проверяется возможность входа, просмотра очереди заявок, изменения статуса и добавления комментария.

### Администратор

Проверяется возможность входа, просмотра статистики, пользователей, категорий и выгрузки отчета.

## Проверка документации

Проверяется наличие:

- README.md;
- инструкции запуска;
- тестовых учетных записей;
- сценария демонстрации;
- чек-листов;
- скриншотов интерфейса.

## Итог

Проект считается готовым к сдаче, если основные сценарии проходят без ошибок, а репозиторий содержит актуальные материалы.
