# Итоговая сводка завершения проекта

Документ фиксирует завершение основного этапа разработки StudentRequest.

## Выполненные работы

В рамках проекта выполнены следующие работы:

- создана структура Flask-приложения;
- реализованы роли студента, сотрудника и администратора;
- настроена авторизация пользователей;
- реализовано создание заявок;
- реализована обработка заявок сотрудником;
- добавлена административная панель;
- добавлена выгрузка XLSX-отчета;
- оформлены основные страницы интерфейса;
- подготовлена справка по системе;
- подготовлена документация проекта;
- добавлены скриншоты интерфейса;
- подготовлена история коммитов;
- обновлен README.md.

## Проверка результата

Перед финальной сдачей необходимо проверить:

- локальный запуск проекта;
- тестовые учетные записи;
- основные пользовательские сценарии;
- наличие скриншотов;
- наличие отчета;
- доступность GitHub-репозитория.

## Текущий результат

Проект StudentRequest подготовлен для использования в отчете по преддипломной практике и демонстрации преподавателю.

## Вывод

Основной этап разработки завершен. Проект содержит исходный код, документацию, скриншоты, тесты и историю изменений в GitHub.
