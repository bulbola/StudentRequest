# Сценарий демонстрации StudentRequest

## Назначение

Данный сценарий предназначен для демонстрации основного функционала веб-информационной системы учета и обработки заявок студентов.

Сценарий показывает полный цикл обработки заявки: от создания обращения студентом до его обработки сотрудником и контроля со стороны администратора.

## Подготовка к демонстрации

Перед началом демонстрации необходимо:

1. Запустить приложение локально.
2. Убедиться, что база данных создана.
3. Убедиться, что тестовые учетные записи добавлены.
4. Открыть приложение в браузере по адресу:

```text
http://127.0.0.1:5000
