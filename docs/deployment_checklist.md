# Чек-лист развертывания проекта

Документ используется для проверки готовности StudentRequest к запуску на другом компьютере или сервере.

## Подготовка окружения

- [ ] Установлен Python.
- [ ] Установлен Git.
- [ ] Склонирован репозиторий проекта.
- [ ] Создано виртуальное окружение.
- [ ] Установлены зависимости из requirements.txt.

## Проверка конфигурации

- [ ] Проверен файл config.py.
- [ ] Проверена переменная SECRET_KEY.
- [ ] Проверен путь к базе данных.
- [ ] Проверена папка для загружаемых файлов.
- [ ] Проверена доступность статических файлов.

## Проверка базы данных

- [ ] Выполнена команда создания базы данных.
- [ ] Выполнена команда добавления тестовых данных.
- [ ] Проверено наличие пользователей.
- [ ] Проверено наличие категорий заявок.
- [ ] Проверено создание новой заявки.

## Проверка запуска

- [ ] Приложение запускается без ошибок.
- [ ] Главная страница открывается.
- [ ] Страница входа открывается.
- [ ] Авторизация работает.
- [ ] Основные роли открывают свои панели.

## Итог

После выполнения данного чек-листа проект считается готовым к демонстрационному запуску.
