# Сценарий устной демонстрации проекта

Документ содержит краткий сценарий устного рассказа при демонстрации StudentRequest.

## Вступление

Проект StudentRequest представляет собой веб-информационную систему учета и обработки заявок студентов в структурном подразделении Университета.

Система предназначена для централизованного приема обращений студентов, их обработки сотрудниками и контроля со стороны администратора.

## Демонстрация главной страницы

На главной странице отображается назначение системы и краткое описание ролей:

- студент;
- сотрудник;
- администратор.

## Демонстрация входа

На странице входа показаны тестовые учетные записи для трех ролей.

Это позволяет быстро проверить работу системы без дополнительной регистрации.

## Демонстрация студента

Под ролью студента демонстрируется:

- личный кабинет;
- список заявок;
- создание новой заявки;
- выбор категории и приоритета;
- просмотр статуса обработки.

## Демонстрация сотрудника

Под ролью сотрудника демонстрируется:

- очередь заявок;
- карточка обработки заявки;
- изменение статуса;
- добавление комментария.

## Демонстрация администратора

Под ролью администратора демонстрируется:

- статистика системы;
- список пользователей;
- категории заявок;
- выгрузка XLSX-отчета.

## Демонстрация GitHub

В репозитории можно показать:

- исходный код проекта;
- структуру папок;
- README;
- документацию;
- скриншоты;
- историю коммитов.

## Заключение

В результате разработки была создана учебная веб-система, реализующая основные функции учета и обработки заявок студентов.
