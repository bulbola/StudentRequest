# Чек-лист проверки GitHub-репозитория

Документ используется для проверки внешнего вида и полноты GitHub-репозитория StudentRequest.

## Общая информация

- [ ] Репозиторий называется StudentRequest.
- [ ] Репозиторий открыт публично.
- [ ] Ветка main содержит актуальный код.
- [ ] README.md отображается на главной странице.
- [ ] В README есть описание проекта.
- [ ] В README есть инструкция запуска.
- [ ] В README есть тестовые учетные записи.

## Структура проекта

- [ ] Есть папка app.
- [ ] Есть папка docs.
- [ ] Есть папка tests.
- [ ] Есть файл requirements.txt.
- [ ] Есть файл run.py.
- [ ] Есть файл wsgi.py.
- [ ] Есть файл .gitignore.

## История разработки

- [ ] Коммиты отражают поэтапное развитие проекта.
- [ ] Есть коммиты по интерфейсу.
- [ ] Есть коммиты по документации.
- [ ] Есть коммиты по исправлениям.
- [ ] Есть коммиты со скриншотами.

## Материалы для отчета

- [ ] В репозитории есть скриншоты.
- [ ] В репозитории есть документация.
- [ ] В отчете указана ссылка на репозиторий.
- [ ] История коммитов может быть показана на защите.
