# Руководство по стилю интерфейса

Документ описывает основные принципы оформления интерфейса веб-информационной системы StudentRequest.

## Общий стиль

Интерфейс системы выполнен в спокойном учебно-деловом стиле. Основной цвет интерфейса — темно-синий. Он используется для верхней панели, кнопок и ключевых элементов управления.

## Основные элементы интерфейса

В системе используются:

- верхняя навигационная панель;
- карточки для группировки информации;
- таблицы для отображения списков;
- кнопки действий;
- бейджи статусов;
- формы ввода данных;
- информационные блоки и подсказки.

## Карточки

Карточки используются для визуального разделения информации.

Примеры использования:

- блоки ролей на главной странице;
- статистика в личном кабинете;
- форма создания заявки;
- карточка обработки заявки;
- административные разделы.

## Таблицы

Таблицы используются для списков:

- заявок студента;
- очереди заявок сотрудника;
- пользователей;
- категорий заявок.

## Формы

Формы должны содержать понятные подписи полей и подсказки.

В системе используются формы:

- входа в систему;
- создания заявки;
- обработки заявки сотрудником.

## Вывод

Единый стиль интерфейса делает систему более понятной для пользователя и упрощает демонстрацию проекта.
