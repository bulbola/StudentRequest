# Заметки по обработке ошибок

Документ описывает возможные ошибки при работе с системой StudentRequest и способы их обработки.

## Ошибки авторизации

Пользователь может ввести неверный email или пароль.

Ожидаемое поведение системы:

- пользователь не получает доступ к личному кабинету;
- система показывает сообщение об ошибке;
- пользователь остается на странице входа.

## Ошибки доступа

Пользователь может попытаться открыть страницу, не предназначенную для его роли.

Ожидаемое поведение системы:

- доступ к странице ограничивается;
- пользователь перенаправляется на доступный раздел;
- система предотвращает выполнение запрещенного действия.

## Ошибки формы заявки

Студент может не заполнить обязательные поля заявки.

Ожидаемое поведение системы:

- форма не отправляется;
- обязательные поля требуют заполнения;
- пользователь может исправить данные.

## Ошибки загрузки файла

При прикреплении файла возможны ошибки:

- файл не выбран;
- файл имеет неподходящее имя;
- файл не может быть сохранен.

Ожидаемое поведение системы:

- заявка создается без файла или выводится сообщение об ошибке;
- путь загрузки файла контролируется приложением.

## Ошибки базы данных

Возможны ситуации, когда база данных не создана или отсутствуют тестовые данные.

Способ устранения:

```powershell
py -m flask --app run.py init-db
py -m flask --app run.py seed-db
