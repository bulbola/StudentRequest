# Финальная проверка качества

Документ описывает итоговую проверку качества проекта StudentRequest перед сдачей.

## Проверка кода

- [ ] Проект запускается без ошибок.
- [ ] Основные маршруты открываются.
- [ ] Формы работают корректно.
- [ ] Страницы не содержат видимых ошибок.
- [ ] CSS-стили подключаются корректно.

## Проверка ролей

- [ ] Студент видит личный кабинет.
- [ ] Сотрудник видит очередь заявок.
- [ ] Администратор видит административную панель.
- [ ] Разделы соответствуют ролям пользователей.

## Проверка документации

- [ ] README содержит описание проекта.
- [ ] README содержит инструкцию запуска.
- [ ] README содержит тестовые учетные записи.
- [ ] В docs есть сценарий демонстрации.
- [ ] В docs есть скриншоты.
- [ ] В docs есть чек-листы.

## Проверка GitHub

- [ ] Репозиторий публичный.
- [ ] История коммитов отображается.
- [ ] Последние изменения отправлены на GitHub.
- [ ] Ветка main актуальна.
- [ ] В репозитории нет лишних временных файлов.

## Проверка отчета

- [ ] В отчете указана тема проекта.
- [ ] В отчете есть ссылка на GitHub.
- [ ] В отчете есть скриншоты интерфейса.
- [ ] В отчете нет лишних служебных следов.
- [ ] Оформление отчета проверено.

## Вывод

После выполнения финальной проверки проект можно использовать для сдачи и демонстрации.
