# Финальный чек-лист демонстрации

Документ используется перед показом проекта StudentRequest преподавателю.

## Перед демонстрацией

- [ ] Проверить доступность GitHub-репозитория.
- [ ] Проверить, что проект запускается локально.
- [ ] Проверить наличие тестовых данных.
- [ ] Проверить работу страницы входа.
- [ ] Проверить тестовые учетные записи.

## Демонстрация студента

- [ ] Войти под студентом.
- [ ] Открыть личный кабинет.
- [ ] Показать список заявок.
- [ ] Открыть форму создания заявки.
- [ ] Создать новую заявку или показать существующую.
- [ ] Показать статус заявки.

## Демонстрация сотрудника

- [ ] Войти под сотрудником.
- [ ] Открыть очередь заявок.
- [ ] Открыть страницу обработки заявки.
- [ ] Изменить статус.
- [ ] Добавить комментарий.
- [ ] Сохранить изменения.

## Демонстрация администратора

- [ ] Войти под администратором.
- [ ] Открыть административную панель.
- [ ] Показать статистику.
- [ ] Открыть пользователей.
- [ ] Открыть категории.
- [ ] Показать выгрузку XLSX.

## Демонстрация GitHub

- [ ] Показать README.
- [ ] Показать структуру проекта.
- [ ] Показать папку docs.
- [ ] Показать историю коммитов.
- [ ] Показать скриншоты интерфейса.

## Итог

После прохождения чек-листа проект готов к демонстрации.
