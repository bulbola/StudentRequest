# Чек-лист тестирования маршрутов

Документ используется для ручной проверки основных маршрутов веб-информационной системы StudentRequest.

## Общие маршруты

- [ ] Главная страница открывается по адресу `/`.
- [ ] Страница входа открывается по адресу `/login`.
- [ ] Страница справки открывается по адресу `/help`.
- [ ] Статические файлы CSS загружаются корректно.
- [ ] Иконка сайта загружается корректно.

## Маршруты студента

- [ ] После входа студент попадает в личный кабинет.
- [ ] Студент может открыть форму создания заявки.
- [ ] Студент может создать заявку.
- [ ] Студент может открыть карточку своей заявки.
- [ ] Студент видит статус заявки и комментарии сотрудника.

## Маршруты сотрудника

- [ ] После входа сотрудник попадает в панель сотрудника.
- [ ] Сотрудник видит очередь заявок.
- [ ] Сотрудник может открыть страницу обработки заявки.
- [ ] Сотрудник может изменить статус заявки.
- [ ] Сотрудник может добавить комментарий.

## Маршруты администратора

- [ ] После входа администратор попадает в панель администратора.
- [ ] Администратор может открыть список пользователей.
- [ ] Администратор может открыть список категорий.
- [ ] Администратор может скачать XLSX-отчет.

## Вывод

Проверка маршрутов подтверждает, что основные страницы системы доступны и соответствуют ролям пользователей.
