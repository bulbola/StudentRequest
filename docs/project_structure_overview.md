# Обзор структуры проекта

Документ описывает структуру проекта StudentRequest и назначение основных папок и файлов.

## Корневая папка

В корневой папке проекта находятся основные файлы запуска, настройки и документации.

## app

Папка содержит основной код Flask-приложения.

В ней находятся:

- модели базы данных;
- маршруты приложения;
- шаблоны страниц;
- статические файлы;
- функции безопасности.

## app/templates

Папка содержит HTML-шаблоны страниц.

Шаблоны разделены по ролям:

- admin;
- auth;
- employee;
- shared;
- student.

## app/static

Папка содержит CSS-стили, favicon и другие статические файлы.

## docs

Папка содержит документацию проекта:

- сценарий демонстрации;
- чек-листы;
- описание ролей;
- описание базы данных;
- скриншоты;
- инструкции запуска.

## tests

Папка содержит тесты проекта.

Тесты используются для проверки доступности страниц и базовой работоспособности приложения.

## requirements.txt

Файл содержит зависимости Python, необходимые для запуска проекта.

## run.py

Файл используется для запуска Flask-приложения и выполнения служебных команд.

## wsgi.py

Файл предназначен для использования приложения через WSGI.

## Вывод

Структура проекта разделяет код, шаблоны, документацию и тесты, что делает проект понятным и удобным для проверки.
