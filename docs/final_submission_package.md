# Финальный комплект для сдачи

Документ описывает состав материалов, которые используются для сдачи проекта по преддипломной практике.

## Основные материалы

В финальный комплект входят:

- отчет по преддипломной практике в формате DOCX;
- ссылка на GitHub-репозиторий;
- исходный код проекта;
- документация проекта;
- скриншоты интерфейса;
- инструкция по локальному запуску;
- тестовые учетные записи.
