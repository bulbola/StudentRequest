# Финальный чек-лист проекта StudentRequest

Документ используется для проверки готовности проекта к сдаче по преддипломной практике.

## Проверка репозитория

- [ ] Репозиторий доступен по ссылке GitHub.
- [ ] Репозиторий является публичным.
- [ ] В репозитории есть README.md.
- [ ] В репозитории есть история коммитов.
- [ ] В репозитории есть документация проекта.
- [ ] В репозитории есть исходный код приложения.
- [ ] В репозитории есть тесты.

## Проверка запуска

- [ ] Создается виртуальное окружение.
- [ ] Устанавливаются зависимости из requirements.txt.
- [ ] Создается база данных.
- [ ] Добавляются тестовые данные.
- [ ] Приложение запускается локально.
- [ ] Главная страница открывается в браузере.

## Проверка ролей

- [ ] Вход под студентом работает.
- [ ] Вход под сотрудником работает.
- [ ] Вход под администратором работает.
- [ ] Для каждой роли отображается свой интерфейс.
- [ ] Доступ к функциям разделен по ролям.

## Проверка функционала

- [ ] Студент может создать заявку.
- [ ] Студент может посмотреть список своих заявок.
- [ ] Сотрудник может открыть очередь заявок.
- [ ] Сотрудник может обработать заявку.
- [ ] Сотрудник может изменить статус заявки.
- [ ] Сотрудник может добавить комментарий.
- [ ] Администратор может открыть список пользователей.
- [ ] Администратор может открыть список категорий.
- [ ] Администратор может скачать отчет XLSX.

## Проверка материалов для отчета

- [ ] Подготовлены скриншоты интерфейса.
- [ ] Подготовлен скриншот GitHub-репозитория.
- [ ] Подготовлен скриншот истории коммитов.
- [ ] В отчете указана ссылка на GitHub.
- [ ] В отчете отсутствуют лишние служебные следы.
