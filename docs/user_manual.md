# Краткая инструкция пользователя

## Студент
1. Войти в систему.
2. Открыть личный кабинет.
3. Нажать «Создать заявку».
4. Выбрать категорию, указать тему, описание и файл.
5. Отправить заявку и отслеживать ее статус.

## Сотрудник
1. Войти в систему.
2. Открыть очередь заявок.
3. Выбрать заявку.
4. Изменить статус и добавить комментарий.
5. Закрыть заявку после обработки.

## Администратор
1. Войти в админ-панель.
2. Управлять пользователями и категориями.
3. Смотреть статистику.
4. Формировать XLSX-отчет.
