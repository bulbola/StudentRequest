# Матрица соответствия требованиям

| Требование | Реализация |
|---|---|
| Клиент-серверная архитектура | Flask-приложение + браузер + БД |
| SQL-БД | SQLAlchemy-модели и SQL-схема |
| 3 роли | student, employee, admin |
| Личный кабинет | `/student/`, `/employee/`, `/admin/` |
| Админ-панель | управление пользователями и категориями |
| Справка | `/help` |
| Доступ к файловой системе | загрузка вложений к заявке |
| Формирование документов | экспорт отчета XLSX |
| 10+ страниц | главная, вход, кабинеты, формы, карточки, справка, отчеты |
| Тестирование | `docs/test_plan.md`, `tests/test_smoke.py` |
