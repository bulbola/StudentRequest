# Модель статусов заявки

| Статус | Назначение |
|---|---|
| created | Заявка создана студентом и ожидает просмотра сотрудником. |
| in_progress | Сотрудник принял заявку в работу. |
| need_info | Для продолжения обработки требуется уточнение от студента. |
| processed | Заявка обработана, результат подготовлен. |
| closed | Заявка закрыта после завершения обработки. |
| rejected | Заявка отклонена, если не относится к компетенции подразделения или заполнена некорректно. |

Основной сценарий демонстрации: `created -> in_progress -> processed -> closed`.
